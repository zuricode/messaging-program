Simple Chat Program
-------------------

This is a very simple Windows CLI client-server program that allows messages to sent between multiple clients and a server.

Functionality:
- Open the Server.
- Open the Clients.
- Send messages!
- When a client is bored, type "exit" and they will be disconnected.
